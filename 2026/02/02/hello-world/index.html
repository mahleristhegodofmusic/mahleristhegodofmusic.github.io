未来如何更新日记？

  1. 写文章：
    在 MyBlog/source/_posts 文件夹里新建或复制进你的 .md 笔记文件（记得加上开头的时间和标题配置）。
  2. 清理旧文件：
    在文件夹右键打开 Git Bash / PowerShell，输入：hexo clean
  3. 生成网页：
    输入：hexo g
  4. 上传发布：
    输入：hexo d

搞定！你的网站现在属于互联网了！如果有任何排版或者美化的问题，欢迎随时回来问我。
